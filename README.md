## movie-seat-booking
Movie Seat Booking App – JavaScript assignment (Frontend 2)

This is an individual JavaScript assignment for the course.
The application allows users to select a movie, choose seats, and see the total price update dynamically.

# Features
- Select a movie from a dropdown list
- Movie data is fetched from a local JSON Server (REST API)
- Click seats to select or unselect them
- Occupied seats cannot be selected
- Seat count and total price update dynamically
- Code is structured using JavaScript

# Technologies Used
- HTML
- CSS
- JavaScript 
- JSON Server (for local development)
